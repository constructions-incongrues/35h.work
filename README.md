# 35h.work

### Pré-requis

- [Virtualbox](http://www.virtualbox.org)
- [Vagrant](http://www.vagrantup.com)

### Installation

```sh
git clone git@github.com:constructions-incongrues/35h.work.git
cd 35h.work
vagrant up
```

### Développement

```sh
vagrant ssh
cd //vagrant/src/wp-content/themes/35h
gulp watch
```
